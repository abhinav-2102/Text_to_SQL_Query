A robust Text-to-SQL application that converts natural language questions into SQL queries using Google's Gemini Pro/Flash models. The application retrieves real-time data from a local SQLite database and displays the results via a user-friendly Streamlit interface.

* **Natural Language Processing:** Converts English questions (e.g., "How many students are in Data Science?") into executable SQL queries.
* **Dynamic Model Selection:** Automatically detects and uses the best available Google Gemini model (e.g., `gemini-1.5-flash` or `gemini-pro`).
* **Database Integration:** Connects to a local SQLite database (`student.db`) to execute queries and fetch results.
* **Schema Visualization:** Displays the database schema (columns and data types) in the sidebar for easy reference.
* **Error Handling:** Robust handling for API errors, SQL syntax errors, and empty datasets.
